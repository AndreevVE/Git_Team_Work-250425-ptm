Cкрипт, который будет выполнять следующую задачу: копировать все файлы с определенным расширением из 
одной директории в другую и изменять расширение на заданное.

1.Запросить исходную и целевую директории у пользователя
Описание: Запросить у пользователя указание исходной и целевой директорий.
Действие: Предложить пользователю ввести пути к исходной и целевой директориям.
Использовать следующие переменне: Исходная директория - source_directory 
Целевая директория - target_directory
2.Запросить расширение файлов для копирования
Описание: Запросить у пользователя указание расширения файлов, которые нужно скопировать.
Действие: Предложить пользователю ввести расширение файлов.
Использовать для сохранения переменную - file_extension 
3.Запросить новое расширение для файлов.
Описание: Запросить у пользователя новое расширения файлов.
Действие: Предложить пользователю ввести расширение файлов.
Использовать для сохранения переменную - new_file_extension 
4.Проверить существование и доступность исходной и целевой директорий
Описание: Проверить, существуют ли и доступны ли исходная и целевая директории.
Действие: Проверить существование и доступность директорий. Если они не существуют или недоступны, вывести сообщение об ошибке.
Прервать работу скрипта с кодом 1
5.Проверить наличие файлов с указанным расширением в исходной директории
Описание: Проверить, есть ли файлы с указанным расширением в исходной директории.
Действие: Поиск файлов с указанным расширением в исходной директории. Если файлы не найдены, вывести сообщение об ошибке.
Прервать работу скрипта с кодом 1
6.Скопировать файлы с новым расширением в целевую директорию
Описание: Скопировать все файлы с указанным расширением из исходной директории в целевую.
Действие: Копирование каждого файла с указанным расширением из исходной директории в целевую. Вывести сообщение о копировании каждого файла.
7.Создание архива исходных файлов.
Описание: Создать сжатый tar архив с исходными файлами.

#!/bin/bash

# Параметры
source_directory="$1"         # Исходная директория
file_extension="$2"          # Расширение файлов
target_directory="$3"        # Целевая директория

# Проверка, что все аргументы переданы
if [ -z "$source_directory" ] || [ -z "$file_extension" ] || [ -z "$target_directory" ]; then
    echo "Использование: $0 <source_directory> <file_extension> <target_directory>"
    exit 1
fi

# Проверка существования исходной директории
if [ ! -d "$source_directory" ]; then
    echo "Ошибка: Исходная директория '$source_directory' не существует."
    exit 1
fi

# Проверка существования целевой директории, создание, если не существует
if [ ! -d "$target_directory" ]; then
    mkdir -p "$target_directory"
fi

# Получение текущей даты в формате Y-M-D
current_date=$(date +%Y-%m-%d)

# Имя архива
archive_name="old_files_${current_date}.tar.gz"

# Создание архива
tar -czf "${target_directory}/${archive_name}" -C "$source_directory" --wildcards "*.$file_extension"

# Проверка успешности создания архива
if [ $? -eq 0 ]; then
    echo "Архив создан: ${target_directory}/${archive_name}"
    # Удаление файлов с указанным расширением из исходной директории
    find "$source_directory" -type f -name "*.$file_extension" -delete
    echo "Файлы с расширением .$file_extension удалены из $source_directory"
else
    echo "Ошибка при создании архива."
    exit 1
fi

Действие: Архивируем файлы из исходной директории source_directory с расширением - file_extension
Архив создается в целефой директории - target_directory
Имя архива: old_files_"Текщая дата в формате Y-M-D".tar.gz
Удалите файлы с расширением file_extension из исходной директории
